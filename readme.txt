Widgit zur Einbindung einer Tabelle des VLW

== Description ==

Dieses Widget zeigt eine aktuelle Tabelle zu einer beliebigen Staffel und Saison des Volleyball-Landesverband Württemberg an. Weiterhin kann eine gewünschte Mannschaft hervorgehoben werden.

Die angezeigten Daten werden von der XML-Schnittstelle des VLW bezogen und in eine HTML Tabelle umgewandelt.

Der VLW bietet unter https://vlw.it4sport.de alle Tabellen als XML Datei. Hierbei wird innerhalb des Widget die URL https://vlw.it4sport.de/data/vbwb/aufsteiger/public/tabelle_[Saison]_[Staffel-ID].xml abgerufen.

Die benötige Staffel ID kann unter http://www.vlw-online.de/index.php?tacoma=webpart.pages.TacomaDynamicPage&navid=6124&coid=6124&cid=1&tacomasid=josvhfeetgbeab0eqtpalkuu62 eingesehen werden.
